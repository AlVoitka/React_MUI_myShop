Часто возникает потребность посмотреть список npm пакетов установленных локально/глобально. Казалось бы легко, 
но не тут то было, простая комманда npm list, показывает лапшу из пакетов и их сотни зависимостей, где ничего не разобрать.

Что бы посмотреть только пакеты, есть команда:

npm list -g --depth=0

или

npx npm list -g --depth=0
…где “-g” - показать глобальные пакеты
