Функция Number

Функция Number преобразует переданное значение в число. В случае строки пробельные символы по краям обрезаются, 
затем, если строку возможно преобразовать в число - то возвращается это число, а если нет - возвращается NaN. 
Значение true преобразуется к числу 1, значение false - к числу 0. Объекты при численных преобразованиях превращаются в NaN.

Синтаксис
Number(что нужно преобразовать);


Пример 
Преобразуем несколько значений в число:

Number(true) // 1
Number(false) // 0

Number('53') // 53
Number(' 5 ') // 5
Number('') // 0
Number('a5') // NaN
Number('5a') // NaN
Number('5 5') // NaN
Number('true') // NaN

Number({}) // NaN
Number([]) // 0
Number([1]) // 1
Number([1, 2]) // NaN
